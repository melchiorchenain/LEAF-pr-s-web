Welcome to LEAF

LEAF is a free social networking platform and donation service that is fully cyber security protected.

The LEAF Goal

The GOAL

Accessibility

LEAF social media takes pride in being accessible to everyone and allowing users to play a key part in the social media platform. We aim for all users to use the network to support their causes freely. LEAF gives access to all features and does not privilege popular users. The LEAF domain is accessible from anywhere in the world.

Respect

LEAF ensures that people using the website are protected from any form of violence or hate speech, as its algorithm will analyze each user's posts and messages. If the algorithm considers the post inappropriate, it will automatically refuse to post it and warn the user of their misconduct. After three warnings, LEAF will automatically ban the user.

Security

LEAF places much importance on guaranteeing users' safety on its media, and all personal information provided is encrypted. It asks permission from users to share their geolocation and status while using the app. The site will also be protected from cyber attacks, so no one should ever encounter an issue while using LEAF.

The Features

Networking

Leaf proposes a networking/discussion forum accessible to all, where users can discuss any topic with individuals from all over the world. The forum will allow users to like, comment on, and share each post.

Event organising

LEAF's primary function is bringing people together, which is why the event organizing system is one of the principal features that LEAF will propose to users. Any user could organize and announce events on LEAF for people to join them.

Donation System

LEAF intends to raise awareness of global issues, which is why it will propose a donation system allowing users to support their chosen NGOs or associations.

The Code

I developed LEAF using HTML, CSS, JAVASCRIPT, PHP, and SQL. I decided to use PHP for its fast, simple, and direct access to the MySQL database. It also links to HTML, which is very convenient when testing LEAF. I used JavaScript to create real-time responses and adapt to all web browsers, websites, and applications, which are crucial to LEAF's functions. I also chose to use the MySQL database as it is flexible, powerful, and quick while remaining accessible and affordable. Finally, CSS is linked to HTML and makes the front end of any website shine.
