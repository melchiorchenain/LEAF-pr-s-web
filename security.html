Security

LEAF puts a lot of importance in cybersecurity

Encryption

LEAF places a lot of importance on protecting each user's personal information, which is why the access passwords to each account are stored in the database and encrypted. Encrypting content has the advantage of making sensitive files inaccessible to outsiders. This way, if the internal network is hacked, the hacker will be unable to access any user's personal information.

Test

I will perform regular Penetration Testings. These tests methodically hack into LEAF's system to expose as many vulnerabilities as possible from multiple vantage points. I will also perform network testing of our host network and network devices. These tests should allow me to identify the critical vulnerabilities being used to launch cyber-attacks.

FireWall

To protect users from any type of cybersecurity attack, I will use a firewall. A firewall is a network security device that monitors and filters incoming and outgoing network traffic based on the security policies I will have previously established.
